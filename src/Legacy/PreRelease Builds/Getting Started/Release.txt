==== Installation ====

Roblox 2019 theme requires stylus to be installed on the users browser (Skip to step 2 if installed).

(Step 1)
Don't have stylus installed?

Get them here :

(Stylus)
https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne

Click the blue  "Add to Chrome" button and wait until installation has completed.

(Step 2)
Once stylus has been successfully installed, go to https://userstyles.world/style/5399/roblox-2019-old-theme and scroll down until you find a blue Install button, click on it. You will be taken to an install page, look for a grey button called Install style and boom you should successfully have installed the theme. For theme patch notes go to stable-branch.
