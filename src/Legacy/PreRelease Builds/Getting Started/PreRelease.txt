==== Installation ====

(The theme requires stylish or stylus extension to work.)

To first start please head to the main roblox website.
Once done click the stylish / stylus icon in the extensions tray in which this dropdown should appear.
(1.png)
Find Write style for:
Under that click on www.roblox.com or roblox.com and a new window should open that should look like this.
(2.png)
Now download the latest css file from prerelease-branch and click Import, here you insert the css by copying and pasting the css in this text box.
(3.png)
Once done your result should look something like this.
(4.png)
Now click the button that says Overwrite style, then click save and you should now have the style applied!
