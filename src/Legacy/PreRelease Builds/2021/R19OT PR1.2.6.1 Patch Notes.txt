(November 16 2021)
Prerelease 1.2.6.1 is now available.

This update makes the following changes :

Icon error, icon in review, icon blocked, icon pending, and icon unknown icons have been changed to the old ones.
Starcode text boxes should now match the sites theme (this includes the error state as well), fixed the add button on the side of it.
Featured / catalog expiration timer backgrounds have their old background colors.
Fixed the limited icons not showing up on certain cards.
Buttons that are next to a search bar button no longer have any border radius.
