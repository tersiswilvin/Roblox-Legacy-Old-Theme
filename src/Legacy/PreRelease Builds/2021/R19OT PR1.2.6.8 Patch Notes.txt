(December 17 2021)
Prerelease 1.2.6.8 is now available.

This update makes the following changes :

Moved a few items from the games section to the main CSS section.
Redid the games carousel arrows CSS.
Removed some extra remove-panel padding that was laying around.

Removed the filter for the animation icons when using btroblox.
Changed the border radius and thumbnails on profiles when using btroblox.
